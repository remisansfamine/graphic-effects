# Features & Usage

## Controls
The controls are made for QWERTY so, switch your keyboard in QWERTY
The player controls are:
- W - Forward
- S - Backward
- A - Left
- D - Right   
- Space - Upward
- Ctrl - Downward
- Left click - Rotate camera
- Right click - Lock camera
- ESCAPE - Unlock camera
